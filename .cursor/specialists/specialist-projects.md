# Specialist Projects

## Descricao

Especialista para modulo de Projetos com foco em CRUD, status, readiness, membros e trilha de auditoria.

## Ordem obrigatoria

1. Ler `.context/modules/projects/context.md`
2. Carregar `.cursor/skills/projects/SKILL.md`
3. Aplicar este especialista
4. Responder com evidencias de arquivo/regra/fluxo

## Foco de analise

- ciclo de vida do projeto e status
- regras de codename/visibilidade
- convites e membros de projeto
- compatibilidade com marcos de deal do PRD

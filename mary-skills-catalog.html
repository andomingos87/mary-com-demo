agents-md-generator
Descrição

Gera estruturas hierárquicas AGENTS.md para codebases com indexação JIT (Just-In-Time). Garante que a documentação de agentes esteja sempre atualizada e sincronizada com a estrutura real do projeto.

Como usar

Use quando precisar criar ou atualizar o arquivo AGENTS.md do projeto, reorganizar a hierarquia de agentes ou sincronizar a documentação com mudanças na estrutura de pastas.

Exemplos de prompt
  • 1"Gere o AGENTS.md a partir da estrutura atual do projeto, incluindo todos os módulos e specialists"
  • 2"Atualize o AGENTS.md para refletir os novos módulos que adicionamos na sprint"
  • 3"Crie uma estrutura AGENTS.md com JIT index para o monorepo, agrupando por domínio"

Audita e mantém a consistência entre as 5 camadas de context engineering do projeto: CLAUDE.md, AGENTS.md, .context/, .cursor/ e .agents/. Detecta conflitos, gaps e inconsistências.

Como usar

Use sempre que fizer mudanças estruturais no projeto, antes de releases, ou periodicamente para garantir que toda a documentação de contexto está alinhada.

Exemplos de prompt
  • 1"Faça uma auditoria completa das 5 camadas de context engineering e liste inconsistências"
  • 2"Verifique se todos os módulos têm context.md, SKILL.md e specialist correspondentes"
  • 3"Compare o AGENTS.md com a estrutura real de pastas e aponte divergências"

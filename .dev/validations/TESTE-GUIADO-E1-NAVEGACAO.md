# Teste Guiado — E1: Fundação de Navegação

**Data:** 06/04/2026
**Status do Épico:** Concluído
**Objetivo:** Validar que menus, rotas e nomenclaturas estão corretos para cada perfil de usuário.

---

## O que é este épico?

O Épico 1 estabeleceu a **base de navegação** da plataforma Mary AI. Isso significa que cada tipo de usuário (Investidor, Ativo e Advisor) possui menus, rotas e labels adequados ao seu perfil. A navegação é o primeiro ponto de contato do usuário com o sistema — se estiver confusa ou incorreta, toda a experiência é prejudicada.

---

## O que você deve conseguir fazer

- Navegar pela plataforma com clareza, sabendo sempre onde está.
- Ver menus organizados de acordo com o seu perfil (Investidor, Ativo ou Advisor).
- Acessar todas as seções principais sem erros ou páginas em branco.
- Identificar nomenclaturas consistentes em todo o sistema (sem termos misturados ou em inglês onde deveria ser português).

---

## Checklist de Validação (10 tarefas)

Marque cada item após testar. Se encontrar problema, descreva brevemente ao lado.

| # | Tarefa | OK? | Observação |
|---|--------|-----|------------|
| 1 | Faça login como **Investidor**. Verifique se o menu lateral exibe os itens na ordem: **Radar → Teses → Feed → Projetos**. | ☐ | |
| 2 | Faça login como **Ativo**. Verifique se o menu lateral exibe os itens na ordem: **MRS → Radar → Feed → [Nome do projeto/codinome]**. | ☐ | |
| 3 | Faça login como **Advisor**. Verifique se o menu lateral exibe os itens na ordem: **Radar → Perfil → Feed → Projetos**. | ☐ | |
| 4 | Clique em cada item do menu e confirme que a página correspondente carrega corretamente (sem erro 404 ou tela em branco). | ☐ | |
| 5 | Verifique se os **nomes dos menus** estão em português e são claros (ex: "Radar" e não "Dashboard", "Teses" e não "Thesis"). | ☐ | |
| 6 | Redimensione a janela do navegador para tamanho de celular. Verifique se o menu se adapta (menu hambúrguer ou equivalente). | ☐ | |
| 7 | Navegue para uma página interna (ex: detalhe de um projeto) e verifique se consegue **voltar ao início** facilmente pelo menu. | ☐ | |
| 8 | Verifique se o **nome da organização** aparece visível no topo ou sidebar. | ☐ | |
| 9 | Se tiver mais de uma organização, verifique se consegue **alternar entre elas** pelo seletor de organização. | ☐ | |
| 10 | Faça logout e login novamente. Confirme que a navegação permanece correta e consistente. | ☐ | |

---

## Como reportar problemas

Para cada item com problema, informe:
1. **Qual item** falhou (número).
2. **O que esperava** ver.
3. **O que viu** de fato.
4. **Screenshot** (se possível).
